# Data Cleaning Portfolio Project

This repository contains SQL scripts and source data used for a data cleaning portfolio project focused on Nashville housing data.

Contents
- `quries.sql` — SQL queries and transformations used to clean and process the dataset.
- `DataCleaning.sql` — additional SQL utilities (if present).
- `Nashville Housing Data for Data Cleaning.csv` — source dataset.

How to use
1. Open the `.sql` files in your SQL client connected to a database named `DataCleaning`.
2. Review queries in `quries.sql` before executing; some DDL/DML statements modify or drop columns.
